# AI-First CRM HCP Module - Architecture

## Overview

This project is an AI-powered Customer Relationship Management (CRM) system designed for Healthcare Professionals (HCP) interaction logging.

The system allows medical representatives to record interactions using:

1. Structured Form
2. AI Conversational Copilot
3. Voice-to-Text Input

## Modules

### 1. Dashboard
- Shows interaction statistics
- Displays AI insights

### 2. Log Interaction
Two modes:

#### Structured Form
Users manually enter:
- Doctor name
- Hospital
- Products
- Discussion
- Follow-up date
- Notes


#### AI Copilot
Users can:
- Type interaction details
- Speak using voice input

AI extracts structured information automatically.


### 3. Interaction History
- View previous interactions
- Search records
- View summaries
